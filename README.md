# Math Game

A browser-based game for practicing addition and subtraction.

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Features

- **Three difficulty levels**: Easy (1–10), Medium (10–50), Hard (50–100)
- **Addition and subtraction** problems
- **Choose problem count**: 5, 10, or 20 problems per round
- **Instant feedback** on each answer
- **Results page** with score percentage and encouraging messages

## Tech Stack

- React 18 with TypeScript
- Vite for building and dev server
- React Router for navigation
